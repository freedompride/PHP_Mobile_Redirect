-= PHP Mobile Redirection =-

This PHP is able to detect mobile users that visit your page.
Not only that , the code will also redirect the user to another page which is mobile built-in page.

Steps:

1- Include the php file on your above your PHP file.

<?php
include 'detection.php';
?>

2- Change the website that you wish the mobile user to redirect to.

Edit detection.php file and change the Location of the redirection.

NOTE:-
It won't work on HTML unless you're using Ajax function and calling the PHP files.
