                             Mobile Redirect PHP
This PHP was designed to detect the user-agent of the user.
If the user is accessing using a mobile device, it will redirect to another URL .

This is an alternative method for responsiveness . It's mainly used in forms.

Tutorial 

Step 1 - Download the PHP and park it in the same directory of the HTML/PHP you want to implement it.
Files:

detection.php
Mobile_Detect.php

Step 2 - Include this PHP code on the pages you want it to detect.
--------------------------
<?php
include 'detection.php';
?>
--------------------------

Step 3 - Edit the detection.php file and change the Location of the redirection URL.
